Как пользоваться (быстро)

1 Сначала зашифровать (из файла keys.txt  по одному ключу в строке):

python encrypt_keys.py -i keys.txt -o my_secrets.json    


2 Расшифровать и вывести в консоль:

python decrypt_keys.py -i my_secrets.json


3 Расшифровать в файл:

python decrypt_keys.py -i my_secrets.json -o keys_out.txt

Важные заметки по безопасности (прочитайте обязательно)

Пароль — единственный секрет: если вы забудете пароль, данные восстановить нельзя. Храните пароль в надежном менеджере паролей.

Не пересылайте пароль вместе с зашифрованным файлом.


scrypt-параметры n=2**14, r=8, p=1 — это разумный компромисс. На очень слабых системах можно уменьшить n, на мощных — увеличить для лучшей стойкости.

Файл создаётся с правами 0600, но облачное/сетевое хранение добавляет риски — лучше хранить в зашифрованном контейнере (VeraCrypt/LUKS) или в менеджере паролей, поддерживающем бинарные/секретные записи.
